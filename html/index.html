Escaping the Matrix

The website documentation of progress



This is a project website I created in order to practice HTML, CSS, JavaScript, and any other programming language that I might need to learn in order to land myself a tech job. The goal for this website is of course for it to be functional and for me to learn new things but my main focus is getting this website to look nice through CSS and JavaScript.


One thing I do hope to avoid with this website is for it to become bloated. I don't want the code to be a massive mess and I don't want the website to load slowly or for it to look like a mess either. The plan is to keep it minimal and elegant but also avoid making it look generic. Hopefully I can find the artistic sweet spot.
